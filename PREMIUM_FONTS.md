# Premium Font Downloads for Typography Settings

The Typography Settings page includes 12 font options. 8 are system fonts (already available), and 4 are premium fonts that need to be purchased/downloaded.

## System Fonts (Free - Already Available)

These fonts should work on most Mac and Windows systems:

1. **Aileron + Foundation Sans** (Default) - Custom fonts already in project
2. **Helvetica Neue** - System font on Mac, Arial fallback on Windows
3. **Futura + Inter** - Available on Mac, Century Gothic fallback
4. **Didot + Inter** - Available on Mac, Bodoni MT fallback
5. **Avenir Next** - Available on Mac, Century Gothic fallback
6. **Garamond + Inter** - System font, Baskerville fallback
7. **Optima + Inter** - System font on Mac, Segoe UI fallback
8. **Bodoni + Inter** - System font, Didot fallback

## Installation Instructions

### For Mac:

1. Download the font files (.otf or .ttf)
2. Double-click the font file
3. Click "Install Font" in Font Book
4. Restart your browser
5. Go to Settings page and select the font

### For Windows:

1. Download the font files (.otf or .ttf)
2. Right-click the font file
3. Select "Install" or "Install for all users"
4. Restart your browser
5. Go to Settings page and select the font

## Budget-Friendly Alternative

If you have an **Adobe Creative Cloud subscription**, you get access to all 4 premium fonts through Adobe Fonts at no extra cost!

1. Go to [Adobe Fonts](https://fonts.adobe.com)
2. Search for the font name
3. Click "Activate fonts"
4. They'll automatically install on your system
5. Restart browser and test in Settings page

## Free Alternatives

If you want to test similar aesthetics without purchasing:

- **Brandon Grotesque** → Use **Montserrat** (free on Google Fonts)
- **Proxima Nova** → Use **Montserrat** (free on Google Fonts)
- **Gotham** → Use **Montserrat** (free on Google Fonts)
- **Freight Display** → Use **Merriweather** (free on Google Fonts)

These are already set as fallback fonts in the CSS!

## Testing Without Installing

The settings page will show fallback fonts if the premium fonts aren't installed:

- Brandon/Proxima/Gotham → Falls back to Montserrat/Helvetica
- Freight Display → Falls back to Georgia

This lets you preview the aesthetic even without the premium fonts.

## Need Help?

If a font isn't showing after installation:

1. Make sure you restarted your browser
2. Try clearing browser cache
3. Check if the font is installed in your system's Font Book/Fonts folder
4. Try the font in another application (Word, Figma) to verify installation
